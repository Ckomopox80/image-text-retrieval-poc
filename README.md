## Описание

Проект посвящён созданию демонстрационной версии поиска изображений по текстовому запросу.

Система получает на вход текстовое описание сцены и пытается найти наиболее релевантное изображение.  
Для решения задачи используются эмбеддинги изображений и текста, а также модель, предсказывающая степень их соответствия.

## Основные идеи

- **Изображения** векторизуются через **ResNet50**
- **Тексты** векторизуются через **SentenceTransformer `all-MiniLM-L6-v2`**
- Для оценки соответствия используется **полносвязная нейросеть**
- Дополнительно обучаются **DummyRegressor** и **Ridge**
- Реализована **юридическая фильтрация** контента, связанного с детьми

## Что делает проект

- загружает и проверяет данные;
- анализирует экспертные и краудсорсинговые оценки;
- строит единый таргет в диапазоне от 0 до 1;
- удаляет изображения с детским контентом;
- извлекает эмбеддинги изображений и текстов;
- обучает несколько моделей;
- сравнивает качество моделей;
- тестирует лучшую модель на случайных запросах;
- показывает дисклеймер вместо запрещённого контента.
